O OceanGuard é um sistema desenvolvido com Arduino para monitorar a temperatura da água do oceano e detectar a proximidade de objetos. O projeto utiliza sensores para coletar dados e alertar quando valores considerados críticos são identificados.

Objetivo

Auxiliar no monitoramento ambiental, fornecendo alertas visuais e sonoros quando a temperatura ultrapassa limites definidos ou quando um objeto é detectado a uma distância próxima.
